# Changelog

All notable changes to the "ImGui Hex Byte Color Picker" extension will be documented in this file.

## [0.1.0] - 2025-12-03

### Added
- Initial release
- Support for ImGui hex byte color format (0xRRGGBBAA)
- Color preview decorations
- VSCode native color picker integration
- Configuration for target languages (default: Lua)
- Automatic format preservation when updating colors

### Features
- Matches 8-digit hex byte colors with 0x prefix
- Parses RRGGBBAA format correctly
- Converts to/from VSCode Color API
- Uppercase hex formatting on output
